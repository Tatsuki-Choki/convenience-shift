# 1月5日のシフト作成テスト計画

## ゴール
デプロイ環境 (`https://convenienceshift.vercel.app/`) において、1月5日のシフトが正常に作成できることを確認する。

## テスト手順
1. **サイトへのアクセス**
   - ブラウザで `https://convenienceshift.vercel.app/` を開く。

2. **ログイン/認証 (必要に応じて)**
   - ログイン画面が表示された場合は、テスト用アカウントでログインするか、新規作成を行う。
   - ※認証が不要な場合はスキップ。

3. **シフト作成画面への遷移**
   - カレンダーまたはシフト作成ボタンから、入力画面へ移動する。

4. **日付選択と入力**
   - 対象日付として **1月5日** を選択する。
   - テスト用のシフトデータ (時間、役割など) を入力する。

5. **保存**
   - 作成ボタン/保存ボタンを押下する。

6. **確認**
   - カレンダー画面またはリスト画面に戻り、1月5日にシフトが表示されていることを確認する。
